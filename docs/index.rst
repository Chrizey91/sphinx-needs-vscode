Sphinx Needs Vscode Documentation
=================================

A Visual Studio Code extension with rich support for the
Sphinx-Needs Python extension (for all actively supported versions of the Python language: >=3.8),
including features such as code completion, providing Needs information on mouse hover,
goto definition, Sphinx-Needs ID selection, and Sphinx-Needs ID generation.
